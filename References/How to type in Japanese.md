## Enabling IME keyboard

IME = Input Method Editor

Recommended good online dictionary to check spelling, especially for カタカナ words: [Jisho](https://jisho.org/).

## Tricky ones

Typing `ん` means typing `nn`

| おな  | おんあ  | おんな   |
| --- | ---- | ----- |
| ona | onna | onnna |

As usual, we say `wa`「わ」but we write `ha`「は」in a sentence.
わたしは　ミカエルです。
